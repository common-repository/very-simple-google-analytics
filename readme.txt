Google analytics plugin

== Description ==

A very simple to use google analytics plugin.

Just set your google analytics ID and you are good to go.

http://www.google.com/analytics

== Installation ==

1. Extract the plugin to your plugins directory.
2. Activate the plugin in the plugins page.
3. Enter the plugin settings page to set your google analytics ID.
